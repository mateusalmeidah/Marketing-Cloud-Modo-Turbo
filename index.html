Marketing Cloud Modo Turbo

1. Journey Builder Express: Se o cliente quer a jornada pronta ontem, o segredo é usar a técnica da jornada express. Arraste, solte, configure, publique e torça para que funcione!
2. Email Studio Turbo: Templates prontos? Pra quê? O lance é fazer tudo do zero e rápido! Jogue o conteúdo no editor, arraste umas imagens, e aperte o botão enviar com fé!
3. Automação Relâmpago: A automação não está funcionando? Vai na fé e no desespero! Duplica aquela automação antiga, troca umas atividades e torça para os dados fluírem!
4. Data Extension Ninja: Não sabe qual Data Extension usar? Crie uma nova no susto! Copie e cole os campos de outras, e pronto! É só começar a usar!
5. SQL de Guerrilha: Sem tempo para otimizações? O jeito é usar um SELECT * FROM mesmo! E reze para o sistema aguentar o tranco!
6. All-in-One Builder: Quando em dúvida, use todos os builders possíveis! E-mail, SMS, Push, Landing Pages. Manda tudo junto e vê o que cola!
7. Testes à La Carte: Esqueça testes exaustivos. Teste um pouquinho ali, outro pouquinho aqui. Se funcionar uma vez, está pronto para rodar em produção. Depois, é só monitorar de longe e torcer para que ninguém perceba se algo der errado.
8. Relatórios Relâmpago: Precisa de um relatório? Copie o relatório antigo, altere algumas datas e filtros e entregue como se fosse novo! O importante é ter algo em mãos rápido e deixar os detalhes para depois.
9. Gestão de Perfis Hardcore: Não sabe a diferença entre um perfil de Marketing e um perfil de Comunicação? Use o perfil padrão para todos e boa sorte! Se alguém reclamar, a gente finge surpresa.
10. Criação de Tarefas à Moda XGH: Planejamento, backlog, e organização de tarefas são muito bons, mas só quando há tempo. Aqui, as tarefas vão direto para o final da fila e o importante é botar a mão na massa!
11. Banco de Dados MacGyver: Dados incompletos ou sem sentido? Não tem problema! Crie regras no SQL para preencher os vazios. Afinal, o que não pode faltar é um bom volume de dados para análise.
12. Revisão Zero: Não perca tempo revisando código, revisando jornada ou e-mails. Publique logo! Se houver algum problema, corrige no ar.
13. Validação Express: Precisa validar? Manda pro cliente e pede feedback. Se não reclamarem, está perfeito!
14. Feedback Pós-Produção: Só se aprende depois que roda em produção. Se der erro, pelo menos já temos uma lição para a próxima.
15. Atualização na Correria: O cliente pediu uma mudança? Entra direto no código, sem revisão, sem testes. A prioridade é atender o cliente no prazo. Se der certo, ponto para o time!
